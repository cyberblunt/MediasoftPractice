# Работа с лямбда-выражениями и аннотациями

## Описание проекта

Этот проект демонстрирует практическое использование лямбда-выражений и аннотаций в Java 8+.

## Структура проекта

### Лямбда-выражения

1. **Printable интерфейс** - функциональный интерфейс для демонстрации простого лямбда-выражения
2. **Predicate** - функциональный интерфейс для проверки условий
3. **Consumer** - функциональный интерфейс для обработки объектов
4. **Function** - функциональный интерфейс для преобразования данных
5. **Supplier** - функциональный интерфейс для генерации данных

### Аннотации и рефлексия

1. **@DeprecatedEx** - кастомная аннотация для указания устаревших элементов
2. **@JsonField** - кастомная аннотация для сериализации в JSON
3. **DeprecatedProcessor** - обработчик для аннотации @DeprecatedEx
4. **JsonSerializer** - сериализатор для преобразования объектов в JSON

## Файлы проекта

- `Main.java` - основной класс с демонстрацией всех примеров
- `Printable.java` - функциональный интерфейс
- `HeavyBox.java` - класс для демонстрации Consumer
- `DeprecatedEx.java` - кастомная аннотация
- `JsonField.java` - аннотация для JSON сериализации
- `DeprecatedProcessor.java` - обработчик аннотации @DeprecatedEx
- `JsonSerializer.java` - JSON сериализатор
- `TestClass.java` - тестовый класс для демонстрации аннотаций

## Демонстрируемые возможности

### Лямбда-выражения

1. **Простое лямбда-выражение** для интерфейса Printable
2. **Predicate** для проверки строк на null и пустоту
3. **Комбинирование Predicate** с помощью метода and()
4. **Сложные проверки строк** с использованием startsWith и endsWith
5. **Consumer** с методом andThen для обработки объектов HeavyBox
6. **Function** для классификации чисел
7. **Supplier** для генерации случайных чисел

### Аннотации и рефлексия

1. **Кастомная аннотация @DeprecatedEx** с параметром message
2. **Обработка аннотаций** через рефлексию
3. **Аннотация @JsonField** для кастомной сериализации
4. **JSON сериализация** с использованием рефлексии

## Запуск программы

```bash
# Сборка проекта
./gradlew build

# Запуск программы
./gradlew run
```

## Ожидаемый вывод

Программа демонстрирует работу всех лямбда-выражений и аннотаций с подробными комментариями и примерами использования.

## Технические детали

- Минимальная версия Java: 8
- Система сборки: Gradle
- Используемые концепции: лямбда-выражения, функциональные интерфейсы, аннотации, рефлексия

## Основные функциональные интерфейсы

- `Predicate<T>` - для проверки условий
- `Consumer<T>` - для обработки объектов
- `Function<T, R>` - для преобразования данных
- `Supplier<T>` - для генерации данных

## Аннотации

- `@DeprecatedEx` - указывает на устаревший элемент с альтернативой
- `@JsonField` - указывает имя поля в JSON представлении
